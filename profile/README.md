## ◈ THE QUESTION

> <em>만약 sitar가 boom-bap의 베이스라인 위에서 인사말이 아니라 <strong>멜로디</strong>를 잡으면 어떻게 되는가?</em>
>
> <em>만약 djembe가 funky의 그루브 안에서 양념이 아니라 <strong>박자 그 자체</strong>를 정의하면?</em>
>
> <em>만약 가야금이 ambient 패드 안에서 1초 인트로가 아니라 <strong>곡의 중심</strong>이면?</em>

대부분의 popular 음악은 세계의 전통악기를 <strong>양념</strong>으로 쓴다.

영화 OST에서 sitar 한 줄은 <em>Indian-ness</em>의 시그널. 후렴 중간의 djembe는 <em>African energy</em>의 시그널. 1초 산조 인트로는 <em>Korean-ness</em>의 시그널. 모두 짧은 인사로 끝난다. 듣는 사람이 "어, 이국적이네" 하고 다시 본 멜로디로 돌아가게 만드는 cue.

라벨 큐는 그 위치를 거부한다.

여기서 세계의 모든 전통악기는 <strong>주연</strong>이고, 현대 장르는 그 주연이 살 수 있는 <strong>세계</strong>다. sitar가 boom-bap을 <em>사용</em>하는 것이지, boom-bap이 sitar를 <em>빌려 쓰는</em> 것이 아니다. djembe가 funk를 <em>지배</em>하는 것이지, funk가 djembe를 <em>체험</em>하는 것이 아니다.

이 차이는 작아 보이지만 모든 미세 결정을 바꾼다 — 어느 악기가 멜로디를 가지는지, 어느 악기가 박자를 정의하는지, mix에서 누가 가장 크게 들리는지, 후렴이 어느 음계 위에 서는지.

세계의 전통악기는 ethnomusicology 박물관에 있는 유물이 아니다. 수백·수천 년의 음악 어휘를 가지고 살아 있는 <em>연주악기</em>다. 라벨 큐는 그 사실에서 출발한다.

<br/>

## ◈ ⊗

라벨 큐의 모든 트랙은 두 요소의 <strong>텐서 곱(tensor product)</strong>으로 표기된다.

```
sitar  ⊗ boom-bap
djembe ⊗ funky
가야금 ⊗ ambient
oud    ⊗ jazz
```

⊗는 단순한 더하기가 아니다.

```
A + B   →   두 개를 나란히 놓는 것
A ⊗ B   →   한 차원이 다른 차원을 곱해 새로운 공간이 열리는 것
```

sitar가 boom-bap의 그루브 위에서 <em>증폭</em>되고, boom-bap이 sitar의 음계 안에서 <em>재정의</em>된다. 두 요소가 자기 정체성을 잃지 않은 채로, <strong>세 번째 무엇</strong>이 된다.

이것이 라벨 큐가 ⊗를 brand mark로 쓰는 이유다. 모든 발매물 표지에, 모든 트랙 페이지에, 이 로고가 적힌다.

<br/>

## ◈ THE GRID

라벨 큐의 공간은 두 차원의 곱이다.

<strong>Modern genres</strong> (6, fixed)

```
boom-bap   ·   funky    ·   jazz
ambient    ·   lo-fi    ·   experimental
```

<strong>World instruments</strong> (open-ended)

```
EAST ASIA       가야금 · 해금 · 장구 · 단소 · 거문고 · 북 · koto · shakuhachi · erhu · pipa
SOUTH ASIA      sitar · tabla · sarod · tampura · santoor · sarangi
WEST ASIA       oud · ney · qanun · darbuka · saz · duduk
AFRICA          kora · djembe · mbira · balafon · talking drum · ngoni
EUROPE          bodhrán · hurdy-gurdy · accordion · uilleann pipes · nyckelharpa
AMERICAS        charango · cajón · banjo · steel drum · native flute · berimbau
OCEANIA         didgeridoo · slit drum · pātē
GLOBAL          hang · glass harmonica · theremin · waterphone
```

이론적으로 6 × N개의 조합이 가능하다. N은 열려 있다. 모든 조합이 발행되지는 않는다 — 살아남는 것만 발행된다.

> <strong>살아남는다는 것</strong> = 두 요소 모두 자기를 잃지 않으면서, 들었을 때 <strong>하나의 곡으로</strong> 들리는 것.

가장 어려운 조합은 보통 가까운 조합이다. <em>해금 ⊗ ambient</em>는 둘 다 sustained tone이라 진부해지기 쉽고, <em>djembe ⊗ funky</em>는 둘 다 percussion이라 뻔해지기 쉽다. 거리가 먼 조합 — <em>단소 ⊗ funky</em>, <em>theremin ⊗ boom-bap</em>, <em>steel drum ⊗ jazz</em> — 이 가장 가능성이 크다.

## ◈ HOW IT WORKS

```
1. GENERATE   →  iq-composer (Suno API 기반) 가 한 트랙당 30~50개 후보 생성
2. CURATE     →  IQ가 듣고, 살아남는 1개만 선택. 통과율 ~3%
3. ANNOTATE   →  vibe notes를 손으로 작성. 왜 이 트랙이 살아남았는지
4. RELEASE    →  CC-BY-NC-4.0으로 발행. 비상업적 사용 자유
```

이 워크플로우가 라벨 큐를 <em>공장</em>이 아닌 <strong>레이블</strong>로 만든다. 양산이 아니라 큐레이션이 본질이다.

> 한 사람 + AI가 운영하는 인디 레이블의 새로운 형태.
> 비용은 0에 가깝고, 심사관은 한 명이며, 모든 결정은 <em>왜?</em> 에 답할 수 있어야 한다.

생성 도구 <strong>iq-composer</strong>는 [iq-agent-lab](https://github.com/iq-agent-lab) 산하에서 개발된다. 한 사람이 6개 도메인의 콘텐츠를 양산·큐레이션할 수 있게 하는 더 큰 시스템(iq-blogger, iq-label, iq-writer, iq-painter, iq-studio, iq-curator)의 두 번째 모듈.
